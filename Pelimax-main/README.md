PROYECTO: PELICULAS EN INTERNET


El proyecto consiste en dos apartados:

•	Aplicación web

•	Aplicación móvil

Aplicación WEB, debe permitir un inicio de sesión y si las claves son correctas, debe permitir el acceso, donde se presentan 4 apartados:

•	Registro de Películas

•	Consulta de peliculas

•	Registro de Clientes

•	Registro de Usuarios

En el apartado de la aplicación móvil se debe permitir iniciar sesión, y poder visualizar las películas activas.
